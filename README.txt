To install database:
	(1) locate the file called 'db.sql' in dbbackup and save the full path
	(2) enter your MariaDB under your prefered user
	(3) type '\. <full path to db.sql>'
	(4) the database should be loaded

The note for the ER diagram are contained within the file called "ER-description.txt"

"createTables.sql" is the file used to create the tables
