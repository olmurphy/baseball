What is this?

This is a web page for you to look up baseball statistics of all professional baseball players. Need to learn how to use it, keep reading. Otherwise, click here to get started.

How do I use it?

To search, you simply type in the playerid of the professional baseball player you want to search. A comprehensive list of all baseball playerid's can be found here.

What is a playerid?

A playerid is simply a unique id to identify each professional baseball player.

Where is the data from?

We are using data provided by Sean Lahman and the likes. Thanks to Sean and all his contributors.
